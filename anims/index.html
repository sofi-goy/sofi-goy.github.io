Visualizador de Campos Vectoriales

Un campo vectorial es una función que, a cada punto de un espacio, le asigna un vector. Esto sirve para describir, por ejemplo, fluidos, donde a cada particula le asignamos una velocidad, dependiendo de su posición.

Este visualizador permite introducir distintas fórmulas para la velocidad de estas párticulas, con fin de entender mejor estas funciones. Se pueden utilizar las operaciones básicas y funciones conocidas: exp(), log(), sin(), cos(), etc.
